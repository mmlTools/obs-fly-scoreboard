# Settings Dialog

- **Server status**: running / stopped (HTTP health check)
- **Port**: choose, then **Restart Server**
- **Hotkeys: how to bind…** opens a quick guide
- **Open Overlay Folder…** opens `BASE/overlay`

The overlay is served from `http://127.0.0.1:<port>/overlay/index.html`.
